# Privacy Policy – Nubra AI Assistant

Last updated: 23-02-2026

## Overview
Nubra AI Assistant is a Chrome extension designed to assist traders and developers by converting, adapting, and explaining trading-related code across different broker APIs.

## Data Collection
This extension does not collect, store, sell, or share any personal user data.

## User Input
Any text or code entered by the user is processed solely to provide the requested functionality, such as code conversion or explanation. User input is not used for tracking, profiling, or advertising.

## Financial and Authentication Data
The extension does not access, collect, or store:
- Financial or payment information
- Broker account credentials
- Authentication details

## Browsing Activity
The extension does not track browsing history, monitor websites, or collect information about user activity across sites.

## Data Storage
Local browser storage may be used to store user preferences or session state. This data remains on the user’s device and is not transmitted externally.

## Third-Party Services
The extension may communicate with backend services to process user requests. These services receive only the minimum data required to fulfill the requested functionality and do not store or reuse user data beyond request processing.

## Changes
This privacy policy may be updated to reflect changes in functionality or legal requirements.
