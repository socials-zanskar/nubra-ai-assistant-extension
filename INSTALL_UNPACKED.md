# Nubra AI Assistant - Unpacked Install

## 1. Download and extract
- Download the release zip: `nubra-ai-extension-<version>.zip`
- Extract it anywhere on your machine.

## 2. Load in Chrome
- Open `chrome://extensions`
- Enable `Developer mode` (top-right)
- Click `Load unpacked`
- Select the extracted extension folder (the one containing `manifest.json`)

## 3. Verify
- The extension should appear in your extensions list.
- Pin it from the Chrome toolbar if needed.
- Open any webpage and click the Nubra floating button.

## Notes
- This unpacked method does not require Chrome Web Store approval.
- You may see a standard Developer Mode warning in Chrome.
